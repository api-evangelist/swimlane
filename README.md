# Swimlane

Swimlane is a security orchestration, automation, and response (SOAR) platform that provides a REST API for managing playbooks, cases, alerts, workflows, and integrating with security tools. The Swimlane Turbine platform offers agentic AI automation for security operations, enabling enterprise security teams and MSSPs to automate threat detection, incident response, and compliance workflows at scale.

## APIs

- **Swimlane REST API** — Core REST API for programmatic access to users, roles, records, applications, playbooks, cases, and alerts. Authenticated via bearer tokens or PATs. Swagger UI available at `/api/swagger` on each instance.
- **Swimlane Turbine API** — Cloud-native Turbine platform API with 500+ connectors and agentic AI automation.
- **Swimlane SCIM API** — SCIM v2 user provisioning API for automated user lifecycle management (Business plans).
